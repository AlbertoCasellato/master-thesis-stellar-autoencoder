# master-thesis-stellar-autoencoder

Master's thesis project: *Autoencoder Interpolator for Stellar Structure Models*

University of Padova, Physics of Data, 2026
Supervisors: G. Costa, G. Iorio, T. Zingales

---

## Overview
This model uses Autoencoders on PARSEC stellar structure profiles (radial distribution of mass, density, temperature, etc).
The goal is to replace the linear interpolation used by the population synthesis code SEVN, speeding up computations while preserving physical accuracy.

## Data
Data are PARSEC stellar evolution tracks and are not included in this repository.
